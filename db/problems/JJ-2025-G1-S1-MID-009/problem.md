## Q
0이 아닌 세 실수 $a, b, c$에 대하여 $\sqrt{a}\sqrt{b} = -\sqrt{ab}$, $\frac{\sqrt{c}}{\sqrt{a}} = -\sqrt{\frac{c}{a}}$일 때, 항상 서로 다른 두 실근을 갖는 이차방정식을 고르면?

## Choices
① $x^2 + bx + c = 0$
② $x^2 - bx - 4a = 0$
③ $x^2 + ax + b = 0$
④ $ax^2 - 2cx + b = 0$
⑤ $bx^2 - ax - c = 0$

## Answer
③

## Solution
주어진 두 조건을 분석하여 $a, b, c$의 부호를 결정합니다.

1.  $\sqrt{a}\sqrt{b} = -\sqrt{ab}$ 조건은 $a \le 0$ 이고 $b \le 0$ 일 때 성립합니다. 문제에서 $a, b$는 0이 아닌 실수이므로 $a < 0$ 이고 $b < 0$ 입니다.

2.  $\frac{\sqrt{c}}{\sqrt{a}} = -\sqrt{\frac{c}{a}}$ 조건은 $c \ge 0$ 이고 $a < 0$ 일 때 성립합니다. 문제에서 $c$는 0이 아닌 실수이므로 $c > 0$ 이고 $a < 0$ 입니다.

두 조건을 종합하면 다음과 같습니다.
$a < 0$
$b < 0$
$c > 0$

이제 각 이차방정식의 판별식 $D$를 조사하여 $D > 0$ 인 경우를 찾습니다. 이차방정식 $Ax^2 + Bx + C = 0$ 이 서로 다른 두 실근을 가지려면 $D = B^2 - 4AC > 0$ 이어야 합니다.

① $x^2 + bx + c = 0$
$D_1 = b^2 - 4(1)(c) = b^2 - 4c$
$b < 0$ 이므로 $b^2 > 0$. $c > 0$ 이므로 $4c > 0$. 따라서 $b^2 - 4c$는 양수일 수도 있고 음수일 수도 있습니다. 예를 들어, $b=-1, c=1$ 이면 $D_1 = (-1)^2 - 4(1) = 1 - 4 = -3 < 0$. 따라서 항상 서로 다른 두 실근을 갖는다고 할 수 없습니다.

② $x^2 - bx - 4a = 0$
$D_2 = (-b)^2 - 4(1)(-4a) = b^2 + 16a$
$b < 0$ 이므로 $b^2 > 0$. $a < 0$ 이므로 $16a < 0$. 따라서 $b^2 + 16a$는 양수일 수도 있고 음수일 수도 있습니다. 예를 들어, $b=-1, a=-1$ 이면 $D_2 = (-1)^2 + 16(-1) = 1 - 16 = -15 < 0$. 따라서 항상 서로 다른 두 실근을 갖는다고 할 수 없습니다.

③ $x^2 + ax + b = 0$
$D_3 = a^2 - 4(1)(b) = a^2 - 4b$
$a < 0$ 이므로 $a^2 > 0$. $b < 0$ 이므로 $-4b > 0$. 따라서 $D_3 = a^2 + (-4b) > 0 + 0 = 0$. 항상 서로 다른 두 실근을 갖습니다.

④ $ax^2 - 2cx + b = 0$
$D_4 = (-2c)^2 - 4(a)(b) = 4c^2 - 4ab = 4(c^2 - ab)$
$c > 0$ 이므로 $c^2 > 0$. $a < 0, b < 0$ 이므로 $ab > 0$. 따라서 $c^2 - ab$는 양수일 수도 있고 음수일 수도 있습니다. 예를 들어, $c=1, a=-1, b=-1$ 이면 $D_4 = 4(1^2 - (-1)(-1)) = 4(1-1) = 0$. 이 경우 중근을 가지므로 서로 다른 두 실근을 갖는다고 할 수 없습니다.

⑤ $bx^2 - ax - c = 0$
$D_5 = (-a)^2 - 4(b)(-c) = a^2 + 4bc$
$a < 0$ 이므로 $a^2 > 0$. $b < 0, c > 0$ 이므로 $bc < 0$. 따라서 $4bc < 0$. $a^2 + 4bc$는 양수일 수도 있고 음수일 수도 있습니다. 예를 들어, $a=-1, b=-1, c=1$ 이면 $D_5 = (-1)^2 + 4(-1)(1) = 1 - 4 = -3 < 0$. 따라서 항상 서로 다른 두 실근을 갖는다고 할 수 없습니다.

따라서 항상 서로 다른 두 실근을 갖는 이차방정식은 ③번입니다.

정답: ③
