## Q
두 다항식 $A, B$에 대하여 $A+B = 4x^2 - x + 5$ 이고 $A-2B = x^2 + 2x - 1$ 일 때, 다항식 $A = ax^2+bx+c$ 에 대하여 $a+b+c$ 의 값을 구하시오.

## Choices
① 6
② 7
③ 8
④ 9
⑤ 10

## Answer
6

## Solution
주어진 두 식은 다음과 같습니다.
1. $A+B = 4x^2 - x + 5$
2. $A-2B = x^2 + 2x - 1$

(1)식에서 (2)식을 빼면 $B$를 구할 수 있습니다.
$(A+B) - (A-2B) = (4x^2 - x + 5) - (x^2 + 2x - 1)$
$A+B-A+2B = 4x^2 - x + 5 - x^2 - 2x + 1$
$3B = 3x^2 - 3x + 6$

양변을 3으로 나누면 $B$는 다음과 같습니다.
$B = x^2 - x + 2$

이제 $B$를 (1)식에 대입하여 $A$를 구합니다.
$A = (4x^2 - x + 5) - B$
$A = (4x^2 - x + 5) - (x^2 - x + 2)$
$A = 4x^2 - x + 5 - x^2 + x - 2$
$A = (4-1)x^2 + (-1+1)x + (5-2)$
$A = 3x^2 + 0x + 3$
$A = 3x^2 + 3$

다항식 $A = ax^2+bx+c$ 이므로, $a=3$, $b=0$, $c=3$ 입니다.
따라서 $a+b+c$ 의 값은 $3+0+3 = 6$ 입니다.
