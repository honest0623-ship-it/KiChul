## Q
서로 다른 세 실수 $p, q, r$이 다음 조건을 만족시킬 때, $pqr$의 값을 구하시오.

(가) $p^3 - 7p^2 + 2p + 31 = p^2 + q^2 + r^2$
(나) $q^3 - 7q^2 + 2q + 31 = p^2 + q^2 + r^2$
(다) $r^3 - 7r^2 + 2r + 31 = p^2 + q^2 + r^2$

## Choices


## Answer
14

## Solution
주어진 세 조건을 살펴보면, $p, q, r$이 모두 동일한 형태의 식을 만족하고 있음을 알 수 있습니다.

$p^3 - 7p^2 + 2p + 31 = p^2 + q^2 + r^2$
$q^3 - 7q^2 + 2q + 31 = p^2 + q^2 + r^2$
$r^3 - 7r^2 + 2r + 31 = p^2 + q^2 + r^2$

여기서 $p^2 + q^2 + r^2$은 상수이므로, 이 값을 $k$라고 합시다.
$k = p^2 + q^2 + r^2$

그러면 위 세 식은 다음과 같이 쓸 수 있습니다.
$p^3 - 7p^2 + 2p + 31 = k$
$q^3 - 7q^2 + 2q + 31 = k$
$r^3 - 7r^2 + 2r + 31 = k$

이는 $p, q, r$이 방정식 $x^3 - 7x^2 + 2x + 31 = k$의 세 근임을 의미합니다. 이 방정식을 정리하면 다음과 같습니다.
$x^3 - 7x^2 + 2x + (31 - k) = 0$

근과 계수의 관계(비에타 정리)에 의해, 이 방정식의 세 근 $p, q, r$에 대해 다음이 성립합니다.
1. $p + q + r = -(-7)/1 = 7$
2. $pq + qr + rp = 2/1 = 2$
3. $pqr = -(31 - k)/1 = k - 31$

이제 $k$ 값을 구하기 위해 $p^2 + q^2 + r^2$의 항등식을 이용합니다.
$p^2 + q^2 + r^2 = (p+q+r)^2 - 2(pq+qr+rp)$

위에서 구한 $p+q+r=7$과 $pq+qr+rp=2$를 대입합니다.
$k = (7)^2 - 2(2)$
$k = 49 - 4$
$k = 45$

구한 $k=45$를 $pqr = k - 31$ 식에 대입합니다.
$pqr = 45 - 31$
$pqr = 14$

따라서 $pqr$의 값은 $14$입니다.

**[참고: 문제 조건의 오류에 대한 설명]**
문제에서 $p, q, r$이 '서로 다른 세 실수'라고 주어졌습니다. 그러나 $k=45$를 대입한 방정식 $x^3 - 7x^2 + 2x - 14 = 0$의 근을 실제로 찾아보면 다음과 같습니다.
$x^3 - 7x^2 + 2x - 14 = x^2(x-7) + 2(x-7) = (x-7)(x^2+2) = 0$
이 방정식의 근은 $x=7$ 또는 $x^2=-2$, 즉 $x=\pm i\sqrt{2}$입니다. 따라서 근은 $7, i\sqrt{2}, -i\sqrt{2}$로, 서로 다른 세 실수가 아닌 하나의 실근과 두 허근을 가집니다. 이는 문제 조건에 모순이 있으나, 일반적으로 이러한 유형의 문제에서는 근과 계수의 관계를 이용하여 값을 구하는 것이 의도된 풀이입니다.
