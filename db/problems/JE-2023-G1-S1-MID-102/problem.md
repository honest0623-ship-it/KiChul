## Q
\[
N=
\frac{(2023+\sqrt{2022})^3+(2023-\sqrt{2022})^3}{2023}
\]
라 할 때, 자연수 \(N\)의 일의 자리 수를 구하시오.

## Choices

## Answer
0

## Solution
\[
a=2023,\quad b=\sqrt{2022}
\]
로 두면
\[
N=\frac{(a+b)^3+(a-b)^3}{a}.
\]

항등식
\[
(a+b)^3+(a-b)^3=2a^3+6ab^2
\]
를 이용하면
\[
N=\frac{2a^3+6ab^2}{a}=2(a^2+3b^2).
\]

따라서
\[
N=2\{2023^2+3\cdot 2022\}.
\]
\[
2023^2=4\,092\,529,\quad 3\cdot 2022=6\,066
\]
이므로
\[
N=2(4\,098\,595)=8\,197\,190.
\]

따라서 \(N\)의 일의 자리 수는
\[
0
\]
이다.
