## Q
이차함수 $f(x)=x^2+ax+b$가 다음 조건을 모두 만족시킬 때, 물음에 답하시오. (단, $a, b$는 상수이다.)
(가) $f(-2)=f(4)$
(나) 함수 $f(x)$의 최솟값은 $-3$이다.
(1) $a, b$의 값을 구하시오.
(2) $-3 \le x \le 6$에서 함수 $f(x)$의 최댓값을 구하시오.

## Choices


## Answer
(1) $a=-2, b=-2$ (2) $22$

## Solution
이차함수 $f(x) = x^2 + ax + b$에 대하여

(1) $a, b$의 값을 구하시오.
조건 (가) $f(-2) = f(4)$에서 이차함수의 대칭축은 $x = \frac{-2+4}{2} = 1$이다.
이차함수 $f(x) = x^2 + ax + b$의 대칭축은 $x = -\frac{a}{2}$이므로,
$-\frac{a}{2} = 1 \implies a = -2$.
따라서 $f(x) = x^2 - 2x + b$이다.
조건 (나) 함수 $f(x)$의 최솟값은 $-3$이다.
$f(x) = x^2 - 2x + b = (x-1)^2 - 1 + b$이므로, 최솟값은 $x=1$일 때 $b-1$이다.
$b-1 = -3 \implies b = -2$.
그러므로 $a = -2, b = -2$.

(2) $-3 \le x \le 6$에서 함수 $f(x)$의 최댓값을 구하시오.
$f(x) = x^2 - 2x - 2$이고, 대칭축은 $x=1$이다.
주어진 범위 $-3 \le x \le 6$에서 대칭축 $x=1$로부터 가장 멀리 떨어진 값에서 최댓값을 갖는다.
$x=1$에서 $-3$까지의 거리는 $|-3-1|=4$이다.
$x=1$에서 $6$까지의 거리는 $|6-1|=5$이다.
따라서 $x=6$일 때 최댓값을 갖는다.
$f(6) = (6)^2 - 2(6) - 2 = 36 - 12 - 2 = 22$.
$f(-3) = (-3)^2 - 2(-3) - 2 = 9 + 6 - 2 = 13$.
최댓값은 $22$이다.
