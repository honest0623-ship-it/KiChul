## Q
$2x^5 - x^4 - 4x^3 - 4x^2 - x + 2 = 0$을 만족하는 실근을 $\alpha$라 할 때, 서로 다른 $\alpha + \frac{1}{\alpha}$의 값을 모두 더하면?

## Choices
① $-\frac{3}{2}$
② $-\frac{1}{2}$
③ $\frac{1}{2}$
④ $\frac{3}{2}$
⑤ $\frac{5}{2}$

## Answer
$\frac{1}{2}$

## Solution
주어진 방정식은 $2x^5 - x^4 - 4x^3 - 4x^2 - x + 2 = 0$ 입니다.
이 방정식은 계수가 대칭인 상반방정식의 형태를 띠고 있습니다. (홀수 차수)
$x=-1$을 대입하면 $2(-1)^5 - (-1)^4 - 4(-1)^3 - 4(-1)^2 - (-1) + 2 = -2 - 1 + 4 - 4 + 1 + 2 = 0$ 이므로 $x=-1$은 실근입니다.
따라서 $(x+1)$로 나누어 인수분해할 수 있습니다.
조립제법을 이용하면:
```
-1 | 2  -1  -4  -4  -1   2
   |    -2   3   1   3  -2
   ---------------------
     2  -3  -1  -3   2   0
```
그러므로 방정식은 $(x+1)(2x^4 - 3x^3 - x^2 - 3x + 2) = 0$ 으로 인수분해됩니다.

첫 번째 실근은 $\alpha = -1$ 입니다. 이 경우 $\alpha + \frac{1}{\alpha} = -1 + \frac{1}{-1} = -1 - 1 = -2$ 입니다.

두 번째 방정식 $2x^4 - 3x^3 - x^2 - 3x + 2 = 0$ 을 풀어야 합니다.
$x=0$은 이 방정식의 근이 아니므로 양변을 $x^2$으로 나눌 수 있습니다.
$2x^2 - 3x - 1 - \frac{3}{x} + \frac{2}{x^2} = 0$
$2(x^2 + \frac{1}{x^2}) - 3(x + \frac{1}{x}) - 1 = 0$
$t = x + \frac{1}{x}$ 로 치환하면 $t^2 = x^2 + 2 + \frac{1}{x^2}$ 이므로 $x^2 + \frac{1}{x^2} = t^2 - 2$ 입니다.
이를 대입하면:
$2(t^2 - 2) - 3t - 1 = 0$
$2t^2 - 4 - 3t - 1 = 0$
$2t^2 - 3t - 5 = 0$
$(2t - 5)(t + 1) = 0$
따라서 $t = \frac{5}{2}$ 또는 $t = -1$ 입니다.

각 경우에 대해 실근 $\alpha$가 존재하는지 확인합니다.
1.  $t = x + \frac{1}{x} = \frac{5}{2}$ 일 때:
    $2x^2 + 2 = 5x$
    $2x^2 - 5x + 2 = 0$
    $(2x - 1)(x - 2) = 0$
    $x = \frac{1}{2}$ 또는 $x = 2$. 이들은 모두 실근입니다.
    이 경우 $\alpha + \frac{1}{\alpha} = \frac{5}{2}$ 입니다.

2.  $t = x + \frac{1}{x} = -1$ 일 때:
    $x^2 + 1 = -x$
    $x^2 + x + 1 = 0$
    이 이차방정식의 판별식 $D = 1^2 - 4(1)(1) = 1 - 4 = -3 < 0$ 이므로 실근이 존재하지 않습니다.
    따라서 이 경우는 문제의 조건인 '실근 $\alpha$'를 만족하지 않습니다.

결론적으로, 실근 $\alpha$에 대해 가능한 서로 다른 $\alpha + \frac{1}{\alpha}$의 값은 $-2$와 $\frac{5}{2}$ 입니다.
이 값들을 모두 더하면:
$-2 + \frac{5}{2} = -\frac{4}{2} + \frac{5}{2} = \frac{1}{2}$

따라서 정답은 $\frac{1}{2}$ 입니다.
