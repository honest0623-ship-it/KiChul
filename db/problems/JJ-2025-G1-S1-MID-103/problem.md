## Q
닫힌 구간 $[2a, 0)$에서 이차함수 $f(x)=ax^2+8ax-10a^2+6$의 최댓값이 6이 되도록 하는 $a$의 값을 구하시오.

## Choices


## Answer
-3/2

## Solution
이차함수 $f(x)=ax^2+8ax-10a^2+6$의 최댓값이 6이 되도록 하는 $a$의 값을 구하시오.
먼저 이차함수를 표준형으로 변형합니다.
$f(x) = a(x^2 + 8x) - 10a^2 + 6$
$f(x) = a(x^2 + 8x + 16 - 16) - 10a^2 + 6$
$f(x) = a(x+4)^2 - 16a - 10a^2 + 6$
이차함수의 축은 $x = -4$입니다.

주어진 구간은 $2a \le x < 0$입니다.

**1. $a > 0$인 경우 (아래로 볼록)**
이차함수가 아래로 볼록하면 최댓값은 구간의 양 끝점에서 발생합니다.
하지만 $2a \le x < 0$에서 $x < 0$이므로 $2a$도 음수여야 합니다.
$a > 0$이면 $2a > 0$이므로 $2a \le x < 0$이라는 구간은 존재할 수 없습니다.
따라서 $a > 0$인 경우는 불가능합니다.

**2. $a < 0$인 경우 (위로 볼록)**
이차함수가 위로 볼록하면 최댓값은 축($x=-4$)에서 발생하거나, 축이 구간 밖에 있을 경우 구간의 끝점에서 발생합니다.

**2.1. 축 $x=-4$가 구간 $[2a, 0)$ 안에 있는 경우**
즉, $2a \le -4 < 0$인 경우입니다.
$2a \le -4$에서 $a \le -2$입니다.
이 경우 최댓값은 축에서 발생하므로 $f(-4)$가 최댓값입니다.
$f(-4) = a(-4)^2 + 8a(-4) - 10a^2 + 6$
$f(-4) = 16a - 32a - 10a^2 + 6$
$f(-4) = -16a - 10a^2 + 6$
주어진 최댓값이 6이므로,
$-16a - 10a^2 + 6 = 6$
$-10a^2 - 16a = 0$
$-2a(5a + 8) = 0$
따라서 $a=0$ 또는 $a = -\frac{8}{5}$입니다.
하지만 이 경우는 $a \le -2$라는 조건에 맞지 않습니다. ($a=0$은 $a<0$에 맞지 않고, $a=-8/5 = -1.6$은 $a \le -2$에 맞지 않습니다.)
따라서 이 경우에서는 해가 없습니다.

**2.2. 축 $x=-4$가 구간 $[2a, 0)$ 밖에 있는 경우 (구간의 왼쪽에 있는 경우)**
즉, $-4 < 2a$인 경우입니다.
$-4 < 2a$에서 $a > -2$입니다.
$a < 0$과 결합하면 $-2 < a < 0$입니다.
이 경우 축 $x=-4$가 구간 $[2a, 0)$의 왼쪽에 있으므로, 함수는 구간 $[2a, 0)$에서 감소합니다.
따라서 최댓값은 구간의 왼쪽 끝점인 $x=2a$에서 발생합니다.
$f(2a) = a(2a)^2 + 8a(2a) - 10a^2 + 6$
$f(2a) = 4a^3 + 16a^2 - 10a^2 + 6$
$f(2a) = 4a^3 + 6a^2 + 6$
주어진 최댓값이 6이므로,
$4a^3 + 6a^2 + 6 = 6$
$4a^3 + 6a^2 = 0$
$2a^2(2a + 3) = 0$
따라서 $a=0$ 또는 $a = -\frac{3}{2}$입니다.
이 경우는 $-2 < a < 0$이라는 조건에 맞아야 합니다.
$a=0$은 조건에 맞지 않습니다.
$a = -\frac{3}{2} = -1.5$는 $-2 < -1.5 < 0$을 만족합니다.
따라서 $a = -\frac{3}{2}$가 유일한 해입니다.

최종적으로 $a = -\frac{3}{2}$입니다.
