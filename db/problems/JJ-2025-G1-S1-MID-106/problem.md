## Q
$-3 \le x \le 3$ 일 때, 이차함수 $f(x)=(2-x)(4+x)+k$ 의 최댓값이 8이다. 이때, $k$의 값을 구하고, 이 함수의 최솟값을 구하시오. (단, $k$는 실수)

## Choices


## Answer
k = -1, 최솟값 = -8

## Solution
주어진 이차함수 $f(x)=(2-x)(4+x)+k$를 전개하면 다음과 같습니다.
$f(x) = (8 + 2x - 4x - x^2) + k$
$f(x) = -x^2 - 2x + 8 + k$
이 이차함수는 최고차항의 계수가 음수이므로 위로 볼록한 포물선입니다. 따라서 꼭짓점에서 최댓값을 가집니다.
꼭짓점의 $x$좌표는 $x = -\frac{-2}{2(-1)} = -1$ 입니다.
주어진 범위 $-3 \le x \le 3$ 안에 $x=-1$이 포함되므로, 최댓값은 $x=-1$일 때 발생합니다.
$f(-1) = -(-1)^2 - 2(-1) + 8 + k = -1 + 2 + 8 + k = 9 + k$
문제에서 최댓값이 8이라고 주어졌으므로,
$9 + k = 8$
$k = -1$

이제 $k=-1$을 함수식에 대입하여 $f(x) = -x^2 - 2x + 8 - 1 = -x^2 - 2x + 7$ 입니다.
최솟값은 위로 볼록한 함수의 경우, 대칭축($x=-1$)으로부터 가장 멀리 떨어진 범위의 끝점에서 발생합니다.
범위의 양 끝점은 $x=-3$과 $x=3$입니다.
$f(-3) = -(-3)^2 - 2(-3) + 7 = -9 + 6 + 7 = 4$
$f(3) = -(3)^2 - 2(3) + 7 = -9 - 6 + 7 = -8$
따라서, 최솟값은 $f(3) = -8$ 입니다.

정답: $k=-1$, 최솟값 $=-8$
