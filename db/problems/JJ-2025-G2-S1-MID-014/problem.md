## Q
함수 $f(x) = \log_a(bx+c) + d$가 $f(10) = 10$, $f(100) = 100$을 만족시킬 때, $f(p)=q$를 만족시키는 두 실수 $p, q$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?
(단, $0 < a < 1$이고, $b, c, d$는 실수)

> <보기>
> ㄱ. $\frac{c}{b} < 0$
> ㄴ. $c \times a^d < 1$
> ㄷ. $10 < p < 100$이면 $q < p$이다.

## Choices
① ㄱ
② ㄱ, ㄴ
③ ㄱ, ㄷ
④ ㄴ, ㄷ
⑤ ㄱ, ㄴ, ㄷ

## Answer
⑤

## Solution
$f(10) = 10$, $f(100) = 100$이므로 함수 $f(x)$는 증가하는 구간을 갖는다.
밑이 $0 < a < 1$이므로 로그함수 $y = \log_a x$는 감소함수이다.
따라서 진수 부분인 $y = bx + c$가 감소해야 전체 함수 $f(x)$가 증가할 수 있으므로 $b < 0$이다.
또한, 진수 조건에 의해 $bx + c > 0$이어야 하므로 $x < -\frac{c}{b}$이다.
정의역에 $x = 10$, $x = 100$이 포함되므로 $100 < -\frac{c}{b}$이다.
따라서 $-\frac{c}{b} > 0$이므로 $\frac{c}{b} < 0$이다. (ㄱ 참)

$f(10) = \log_a(10b+c) + d = 10$ 이므로
$10b+c = a^{10-d} \implies 10ba^d + c a^d = a^{10} \quad \cdots (1)$

$f(100) = \log_a(100b+c) + d = 100$ 이므로
$100b+c = a^{100-d} \implies 100ba^d + c a^d = a^{100} \quad \cdots (2)$

$(2) - (1)$을 하면 $90ba^d = a^{100} - a^{10}$이므로 $ba^d = \frac{a^{100} - a^{10}}{90}$이다.
이를 $(1)$에 대입하면
$ca^d = a^{10} - 10\left(\frac{a^{100} - a^{10}}{90}\right)$
$= \frac{10a^{10} - a^{100}}{9}$ 이다.

$t = a^{10}$이라 하면 $0 < a < 1$이므로 $0 < t < 1$이다.
$t^{10} - 10t + 9$
$= (t-1)(t^9 + t^8 + \cdots + t - 9)$
로 인수분해 할 수 있다.
$0 < t < 1$이므로 $t-1 < 0$이고, $(t^9 + t^8 + \cdots + t) - 9 < 9 - 9 = 0$이다.
두 음수의 곱은 양수이므로 $t^{10} - 10t + 9 > 0$이 성립한다.
따라서 $10t - t^{10} < 9$이고 양변을 $9$로 나누면 $\frac{10t - t^{10}}{9} < 1$이 되어 $ca^d < 1$이다. (ㄴ 참)

$0 < a < 1$일 때, 함수 $y = \log_a x$의 그래프는 아래로 볼록한 모양이다.
따라서 이를 대칭이동 및 평행이동한 함수 $y = f(x) = \log_a(bx+c) + d$의 그래프 역시 아래로 볼록하다.
아래로 볼록한 함수의 그래프는 임의의 두 점을 이은 선분보다 항상 아래쪽에 위치한다.
$f(10) = 10$, $f(100) = 100$이므로 점 $(10, 10)$과 $(100, 100)$을 잇는 선분은 직선 $y=x$의 일부분이다.
따라서 $10 < p < 100$인 $p$에 대하여 함수 $f(x)$의 그래프는 직선 $y=x$보다 아래쪽에 있으므로 $f(p) < p$, 즉 $q < p$가 성립한다. (ㄷ 참)

그러므로 옳은 것은 ㄱ, ㄴ, ㄷ이다.
