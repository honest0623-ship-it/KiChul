## Q
$n^4 - 63n^2 + 49$의 값이 소수가 되도록 하는 자연수 $n$의 값을 $a$, 그때의 소수를 $p$라 할 때, $a+p$의 값은?

## Choices
① 117
② 118
③ 119
④ 120
⑤ 121

## Answer
121

## Solution
주어진 식 $n^4 - 63n^2 + 49$를 인수분해합니다.

$n^4 - 63n^2 + 49 = (n^4 - 14n^2 + 49) - 49n^2$
$= (n^2 - 7)^2 - (7n)^2$
합차 공식을 이용하여 인수분해하면:
$= (n^2 - 7 - 7n)(n^2 - 7 + 7n)$
$= (n^2 - 7n - 7)(n^2 + 7n - 7)$

이 값이 소수가 되려면 두 인수 중 하나는 1이어야 합니다. (자연수 $n$에 대해 두 인수는 양수여야 합니다.)

**경우 1: $n^2 - 7n - 7 = 1$**
$n^2 - 7n - 8 = 0$
$(n-8)(n+1) = 0$
$n$은 자연수이므로 $n=8$입니다.

이때 다른 인수의 값은 $n^2 + 7n - 7 = 8^2 + 7(8) - 7 = 64 + 56 - 7 = 120 - 7 = 113$입니다.
따라서 식의 값은 $1 \times 113 = 113$입니다. $113$은 소수이므로 이 경우는 유효합니다.
이 경우 $a=n=8$이고, $p=113$입니다.
구하는 값 $a+p = 8+113 = 121$.

**경우 2: $n^2 + 7n - 7 = 1$**
$n^2 + 7n - 8 = 0$
$(n+8)(n-1) = 0$
$n$은 자연수이므로 $n=1$입니다.

이때 다른 인수의 값은 $n^2 - 7n - 7 = 1^2 - 7(1) - 7 = 1 - 7 - 7 = -13$입니다.
따라서 식의 값은 $(-13) \times 1 = -13$입니다. 소수는 1보다 큰 자연수이므로 $-13$은 소수가 아닙니다. 따라서 이 경우는 유효하지 않습니다.

유효한 경우는 $n=8$일 때이며, 이때 $a=8$, $p=113$입니다.
따라서 $a+p = 8+113 = 121$입니다.

정답은 ⑤ 121입니다.
