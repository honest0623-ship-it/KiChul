## Q
함수 $f(n)=\left(\frac{1-i}{1+i}\right)^{2n}$, $g(m)=\left(\frac{1+i}{1-i}\right)^{m+1}$ 일 때, $f(1)+g(1)+f(2)+g(2)+\cdots+f(13)+g(13)$의 값은?

## Choices
① $-1-i$
② $1-i$
③ $1+i$
④ $0$
⑤ $-2$

## Answer
⑤

## Solution
먼저 주어진 복소수 식을 간단히 합니다.
$\frac{1-i}{1+i} = \frac{(1-i)(1-i)}{(1+i)(1-i)} = \frac{1-2i+i^2}{1-i^2} = \frac{1-2i-1}{1+1} = \frac{-2i}{2} = -i$
$\frac{1+i}{1-i} = \frac{(1+i)(1+i)}{(1-i)(1+i)} = \frac{1+2i+i^2}{1-i^2} = \frac{1+2i-1}{1+1} = \frac{2i}{2} = i$

따라서 함수 $f(n)$과 $g(m)$은 다음과 같이 간단히 표현됩니다.
$f(n) = (-i)^{2n} = ((-i)^2)^n = (-1)^n$
$g(m) = i^{m+1}$

구하고자 하는 값은 $f(1)+g(1)+f(2)+g(2)+\cdots+f(13)+g(13)$ 입니다.
이 식은 $f$값들의 합과 $g$값들의 합으로 나누어 생각할 수 있습니다.
즉, $(f(1)+f(2)+\cdots+f(13)) + (g(1)+g(2)+\cdots+g(13))$ 을 구하면 됩니다.

1. $f(1)+f(2)+\cdots+f(13)$ 계산:
$f(n) = (-1)^n$ 이므로,
$f(1) = -1$
$f(2) = 1$
$f(3) = -1$
$\cdots$
$f(13) = -1$
이들을 차례대로 두 개씩 묶어서 더하면 $f(1)+f(2) = -1+1=0$ 입니다.
총 13개이므로 두 개씩 6쌍이 짝지어지고, 마지막 항인 $f(13)$ 하나가 남습니다.
따라서 이 합은 $0 \times 6 + f(13) = -1$ 입니다.

2. $g(1)+g(2)+\cdots+g(13)$ 계산:
$g(m) = i^{m+1}$ 이므로,
$g(1) = i^{1+1} = i^2 = -1$
$g(2) = i^{2+1} = i^3 = -i$
$g(3) = i^{3+1} = i^4 = 1$
$g(4) = i^{4+1} = i^5 = i$
$g(5) = i^{5+1} = i^6 = -1$
복소수 $i$의 거듭제곱은 4를 주기로 반복되므로, $g(m)$의 값도 4개씩 반복됩니다.
$g(1)+g(2)+g(3)+g(4) = -1 + (-i) + 1 + i = 0$
총 13개의 항이 있으므로, 4개씩 3묶음이 생기고 하나가 남습니다.
즉, $(g(1)+g(2)+g(3)+g(4)) \times 3 + g(13)$ 과 같습니다.
$13 = 4 \times 3 + 1$ 이므로 $g(13)$은 첫 번째 값인 $g(1)$과 같습니다.
$g(13) = i^{14} = (i^4)^3 \times i^2 = 1^3 \times (-1) = -1$
따라서 이 합은 $0 \times 3 + (-1) = -1$ 입니다.

3. 최종 합 계산:
전체 식의 값은 $-1 + (-1) = -2$ 입니다.

따라서 구하는 값은 $-2$입니다.
