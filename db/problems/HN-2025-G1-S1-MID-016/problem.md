## Q
이차방정식 $2x^2 + 4x + 3 = 0$의 두 근을 $\alpha$와 $\beta$라 할 때, 두 수 $\alpha + \beta$, $\alpha\beta$를 두 근으로 하고 $x^2$의 계수가 1인 이차방정식의 일차항의 계수는?

## Choices
① $-\frac{3}{2}$
② $-\frac{1}{2}$
③ $\frac{1}{2}$
④ $\frac{3}{2}$
⑤ $\frac{5}{2}$

## Answer
$\frac{1}{2}$

## Solution
주어진 이차방정식 $2x^2 + 4x + 3 = 0$의 두 근을 $\alpha$, $\beta$라고 할 때, 근과 계수의 관계에 의해
$\alpha + \beta = -\frac{4}{2} = -2$
$\alpha\beta = \frac{3}{2}$

새로운 이차방정식의 두 근은 $\alpha + \beta$와 $\alpha\beta$이므로, 이 두 근을 각각 $r_1, r_2$라고 하면
$r_1 = \alpha + \beta = -2$
$r_2 = \alpha\beta = \frac{3}{2}$

새로운 이차방정식의 두 근의 합 $S$와 곱 $P$를 구합니다.
$S = r_1 + r_2 = (\alpha + \beta) + (\alpha\beta) = -2 + \frac{3}{2} = -\frac{4}{2} + \frac{3}{2} = -\frac{1}{2}$
$P = r_1 \cdot r_2 = (\alpha + \beta)(\alpha\beta) = (-2) \cdot \left(\frac{3}{2}\right) = -3$

$x^2$의 계수가 1인 이차방정식은 $x^2 - Sx + P = 0$의 형태로 나타낼 수 있습니다.
구한 $S$와 $P$ 값을 대입하면:
$x^2 - \left(-\frac{1}{2}\right)x + (-3) = 0$
$x^2 + \frac{1}{2}x - 3 = 0$

이 이차방정식의 일차항의 계수는 $\frac{1}{2}$입니다.
