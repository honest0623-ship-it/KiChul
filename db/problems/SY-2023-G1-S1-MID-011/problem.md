## Q
복소수
\[
z=(a+1)+(a-9)i
\]
에 대하여
\[
z^2+(\overline{z})^2\le 0
\]
을 만족하는 자연수 \(a\)의 개수는?

## Choices
① 2
② 3
③ 4
④ 5
⑤ 6

## Answer
③

## Solution
\[
z=x+yi\quad (x=a+1,\ y=a-9)
\]
로 두면
\[
z^2+(\overline{z})^2
=(x+yi)^2+(x-yi)^2
=2(x^2-y^2).
\]
따라서 조건은
\[
x^2-y^2\le 0
\]
즉
\[
(a+1)^2\le (a-9)^2.
\]
이를 정리하면
\[
(a+1)^2-(a-9)^2\le 0
\]
\[
20(a-4)\le 0
\]
\[
a\le 4.
\]
자연수 \(a\)는 \(1,2,3,4\)이므로 개수는 \(4\)개이다.
정답은 ③이다.
