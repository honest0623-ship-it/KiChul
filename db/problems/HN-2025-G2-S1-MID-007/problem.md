## Q
함수 $f(x) = -9^x + 2 \times 3^{x+2} + a$가 $x=b$에서 최댓값 $60$을 가질 때, $b-a$의 값은?

## Choices
① $21$
② $22$
③ $23$
④ $24$
⑤ $25$

## Answer
③

## Solution
$f(x) = - (3^x)^2 + 2 \times 9 \times 3^x + a = - (3^x)^2 + 18 \cdot 3^x + a$
$3^x = t$ ($t > 0$)라 하면,
$f(t) = -t^2 + 18t + a = -(t - 9)^2 + 81 + a$ 이다.

$t > 0$ 이므로 $t = 9$ 일 때 최댓값 $81 + a$ 를 가진다.
최댓값이 $60$이므로 $81 + a = 60$, 즉 $a = -21$이다.

이때 $t = 3^x = 9$ 이므로 $x = 2$이다. 따라서 $b = 2$이다.

구하는 값은 $b - a = 2 - (-21) = 23$이다.
