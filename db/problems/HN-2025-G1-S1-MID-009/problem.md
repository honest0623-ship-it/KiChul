## Q
다항식 $x^4 + 2x^3 - 5x^2 - 2x + 4$을 인수분해 하면 $(x-1)(x+1)(x^2+ax+b)$이다. 이때 상수 $a+b$의 값은?

## Choices


## Answer
-2

## Solution
주어진 다항식 $P(x) = x^4 + 2x^3 - 5x^2 - 2x + 4$가 $(x-1)(x+1)(x^2+ax+b)$로 인수분해된다고 했습니다.

먼저 $(x-1)(x+1)$을 계산하면 $x^2 - 1$이 됩니다.

따라서 $P(x) = (x^2 - 1)(x^2+ax+b)$ 입니다.

우변을 전개하면:
$(x^2 - 1)(x^2+ax+b) = x^2(x^2+ax+b) - 1(x^2+ax+b)
$= x^4 + ax^3 + bx^2 - x^2 - ax - b
$= x^4 + ax^3 + (b-1)x^2 - ax - b

이 식을 원래 다항식 $x^4 + 2x^3 - 5x^2 - 2x + 4$와 계수를 비교합니다.

1.  $x^3$의 계수 비교:
    $a = 2$

2.  $x^2$의 계수 비교:
    $b-1 = -5 \Rightarrow b = -4$

3.  $x$의 계수 비교:
    $-a = -2 \Rightarrow a = 2$ (일치)

4.  상수항 비교:
    $-b = 4 \Rightarrow b = -4$ (일치)

따라서 상수 $a=2$이고 $b=-4$ 입니다.

구하고자 하는 값은 $a+b$이므로:
$a+b = 2 + (-4) = -2$

정답은 $-2$입니다.
